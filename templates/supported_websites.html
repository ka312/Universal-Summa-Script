Supported Websites

Universal Summa Script supports a wide range of video and audio platforms. Here's a comprehensive list of supported websites and their features:

Video Platforms
  • YouTube

    Supports all public videos and playlists

  • Vimeo

    Supports public and unlisted videos

  • Dailymotion

    Supports all public videos

  • Facebook

    Supports public video posts

Audio & Podcast Platforms
  • Spotify

    Supports podcast episodes (requires authentication)

  • SoundCloud

    Supports public tracks and podcasts

  • Apple Podcasts

    Supports public podcast episodes

  • Google Podcasts

    Supports all public episodes

Meeting Platforms
  • Zoom

    Supports cloud recordings (requires download link)

  • Microsoft Teams

    Supports meeting recordings (requires SharePoint link)

  • Google Meet

    Supports recordings saved to Google Drive

File Support
Supported File Formats:
  • MP4
  • MP3
  • WAV
  • M4A
  • WEBM
  • OGG
  • FLAC
Note:
  • Maximum file size: 1GB
  • For larger files, please use a direct URL from supported platforms
  • All files are processed securely and deleted after processing
